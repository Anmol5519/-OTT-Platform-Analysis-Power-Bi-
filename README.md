# OTT Platform Content Analysis Dashboard (Power BI)

## Project Overview
This project analyzes content available on major OTT platforms:
- Netflix
- Amazon Prime Video
- Disney+ Hotstar

The dashboard provides insights about movies, series, ratings, duration and yearly content release trends.

## Tools Used
- Power BI
- SQL
- Excel
- Data Cleaning

## Key Insights

1. Netflix contains the highest number of shows among the three platforms.
2. Movies dominate OTT platforms compared to TV series.
3. Most shows fall in the 1–2 hour duration category.
4. TV-MA and PG-13 are the most common ratings.
5. Content releases increased significantly after 2015.
